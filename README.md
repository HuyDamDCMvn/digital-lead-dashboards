# Digital Lead Dashboards (GitHub Pages)

Public static snapshots of Digital Team dashboards generated by
[DigitalLeadAssistant](https://github.com/HuyDamDCMvn/DigitalLeadAssistant).

**Entry point:** [dashboard_control_center.html](dashboard_control_center.html)

## Notes

- Data is a **point-in-time snapshot** from Mantis/MariaDB builds on the publisher machine.
- Munich RE dashboards (BEP/LOIN and Digital Team) and Revizto exports are intentionally excluded.

## Republish (maintainers)

From the DigitalLeadAssistant workspace — refresh local HTML and push Pages in one step:

```powershell
python scripts/refresh_all_dashboards.py --publish
```

Preview publish without git push:

```powershell
python scripts/refresh_all_dashboards.py --publish --publish-dry-run
```

Publish only (after a prior refresh):

```powershell
python scripts/publish_dashboards_pages.py --push
```
